react-three-fiber

一款用 react 的开发方式开发 threeJS。一切在 Threejs 中有效的东西都能使用 React 的编码方式实现。并且得益于 React 的调度能力，它在规模上表现优于 Threejs。通过 JSX 的形式标示 threeJS，实际的转换过程<mesh /> 会动态转换成 new THREE.Mesh()。因此 threejs 版本添加、删除或更改功能，它将立即可用。
